# **App Name**: LocalLens

## Core Features:

- Interactive Place Discovery: Allows users to search for places across India using dynamic search queries (e.g., 'cafes in Kolkata', 'parks in Delhi') and view results in real-time.
- Dual Discovery Modes: Users can toggle between 'Tourist Mode' for popular, highly-rated spots and 'Hidden Gems Mode' for calm, less crowded, local favorites, with results updating instantly.
- Intelligent Hidden Gem Filtering: An AI-powered tool classifies places for 'Hidden Gems Mode' by analyzing descriptions for keywords like 'quiet', 'local', 'peaceful', contributing to an 'Authenticity Score'.
- Integrated Map Visualization: Displays discovered locations on an interactive map (e.g., using Leaflet), featuring custom markers for each place.
- Rich Place Cards: Each search result is presented in a detailed card showing the place's name, image, short description, Crowd Level (Low/Medium/High), and Authenticity Score.
- Local Data Management: Manages a mock dataset of diverse Indian places (cafes, parks, hidden spots), simulating external API interactions without needing a backend database for MVP.
- Dynamic UI Experience: Implements a modern user interface featuring a floating search bar, a bottom sliding panel for results, and smooth animations with transitions for an engaging experience.

## Style Guidelines:

- Color Scheme: Dark theme, evoking calm discovery and a premium feel.
- Primary Color: Muted Teal Blue (#2E5CA6). Chosen to convey depth, peace, and discovery, it contrasts subtly with the dark background while maintaining a sophisticated vibe.
- Background Color: Deep Charcoal Blue (#191D24). A very dark, slightly desaturated blue variant of the primary hue, providing a soft, rich backdrop for the glassmorphism effects.
- Accent Color: Vibrant Aqua (#50CBEE). An analogous hue to the primary, it adds a touch of brightness and serves for interactive elements, highlights, and subtle accents.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, slightly techy, and clean look that enhances readability. Body Text: 'Inter' (sans-serif) for its neutral, objective feel and excellent legibility across various screen sizes.
- Use minimalist, line-based icons with subtle transparency to complement the glassmorphism design, ensuring they are clear and functional against blurred backgrounds.
- Implement a modern glassmorphism design featuring blurred backgrounds and transparent elements, particularly for panels and cards. The main search bar will float at the top, and results will be presented in a bottom-sliding panel for an intuitive mobile-first feel.
- Incorporate smooth and subtle animations for transitions between discovery modes, opening/closing the results panel, and hover effects on place cards to enhance user engagement and visual richness.
